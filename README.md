## Project Overview
This project analyzes a South African soccer dataset using Google BigQuery.
The objective is to answer key analytical questions related to players, teams,
contracts, salaries, and performance metrics.

## Dataset
Source: South African Soccer Dataset  
Tool Used: Google BigQuery  
Table: soccer-project-480914.Soccer.SoccerSet

## Key Questions Answered
- Total number of players
- Players per team
- Top goal scorers
- Average salary per team
- Contract expiry analysis
- Goals and assists contribution

## Tools & Technologies
- Google BigQuery
- SQL

## Key Insights
- Some teams have higher average salaries than others
- Several player contracts expire next season
- Goal contributions are concentrated among a few players
